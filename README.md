## Nintendo Switch

### Change DNS Settings
1. Go to System Settings
2. Internet
3. Find and select the WIFI network your switch is connected to
4. select change settings
5. go to DNS settings - Choose Manual
Primary DNS: 104.238.130.180
Secondary DNS: 008.008.008.008
6. Save Settings
2. Open Minecraft
3. Add server
